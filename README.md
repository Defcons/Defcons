# Martin Davidsen

Self-taught software engineer based in **Stavanger, Norway** 🇳🇴. I build full-stack web
applications, developer tooling, and self-hosted infrastructure — and I run a homelab I maintain a
little too lovingly.

A lot of my work is an ongoing experiment in **AI-assisted ("agentic") software engineering** —
directing AI coding agents to design, build, and ship real, production-grade software end to end.

## 🔭 Featured project

- **[clouddrive](https://github.com/Defcons/clouddrive)** — self-hosted file storage with a **Go**
  backend and a **React + TypeScript** frontend: authentication & MFA, share links, granular
  permissions, trash/restore, and audit logging, all packaged with **Docker**. A from-scratch,
  full-stack app built to run on my own infrastructure.

I spend most of my spare time on **self-hosting and home infrastructure** — running services on a
Proxmox/Docker homelab, automating deployments, and building small tools to tie it all together.
