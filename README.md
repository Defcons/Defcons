# Martin Davidsen

Self-taught software engineer based in **Stavanger, Norway** 🇳🇴. I build full-stack web apps,
developer tooling, in-game addons, and a homelab I maintain a little too lovingly.

A lot of my open-source work is an ongoing experiment in **AI-assisted ("agentic") software
engineering** — directing AI coding agents to build and ship real, production-grade software.
